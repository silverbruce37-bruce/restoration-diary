# 회복과 재건의 묵상일기 - 에스라-느헤미야 시대

에스라-느헤미야 시대의 말씀을 통해 하나님의 경륜을 묵상하고, 포로귀환 시대의 회복과 재건의 이야기를 묵상하는 신앙 성장 앱입니다.

## 📖 읽기 계획

이 앱은 포로귀환 시대와 관련된 6권의 성경을 역사적 순서로 배열하여 매일 2장씩 읽습니다:

| 순서 | 성경 | 장수 | 시대적 배경 |
|------|------|------|-------------|
| 1 | 학개서 | 2장 | BC 520년, 성전 재건 촉구 |
| 2 | 스가랴서 | 14장 | BC 520-518년, 성전 재건과 메시아 예언 |
| 3 | 에스라서 | 10장 | BC 538-458년, 포로 귀환과 성전 완공 |
| 4 | 에스더서 | 10장 | BC 483-473년, 바사 제국에서의 유대인 구원 |
| 5 | 느헤미야서 | 13장 | BC 445년 이후, 성벽 재건과 공동체 회복 |
| 6 | 말라기서 | 4장 | BC 430년경, 언약 갱신 촉구 |

**총 53장**, 매일 2장씩 읽으면 **27일** 완독

## ✨ 주요 기능

### 1. 읽기 & 묵상
- 매일 정해진 2장의 성경 본문 표시
- AI 기반 묵상 가이드 생성
- 하나님의 경륜(섭리) 분석
- 역사적 배경 설명
- 전체 읽기 일정 확인

### 2. 신앙 일기
- 회개와 감사 기록
- 결단과 적용 (재건) 작성
- 하나님이 주신 비전 기록
- 일기 저장 및 조회

### 3. 전도 & 선교
- AI 기반 전도 훈련 팁
- 에스라/느헤미야 스타일 기도문 생성
- 전도 및 선교 계획 작성
- 계획 저장 및 조회

## 🛠️ 로컬 실행

**필수 조건:** Node.js

1. 의존성 설치:
   ```bash
   npm install
   ```

2. `.env.local` 파일에 Gemini API 키 설정:
   ```
   GEMINI_API_KEY=your_api_key_here
   ```

3. 앱 실행:
   ```bash
   npm run dev
   ```

## 📚 역사적 배경

이 앱에서 다루는 시대는 바벨론 포로에서 돌아온 유대인들의 회복 이야기입니다:

- **BC 586년**: 예루살렘 함락, 바벨론 포로
- **BC 538년**: 고레스 칙령으로 1차 귀환 (스룹바벨 인솔)
- **BC 520-516년**: 성전 재건 (학개, 스가랴 선지자 활동)
- **BC 483-473년**: 에스더의 이야기
- **BC 458년**: 에스라의 2차 귀환
- **BC 445년**: 느헤미야의 3차 귀환, 성벽 재건
- **BC 430년경**: 말라기 선지자 활동

## 🙏 묵상 포인트

1. **회복의 하나님**: 포로에서 돌아온 백성들을 통해 역사하시는 하나님
2. **재건의 사명**: 성전과 성벽 재건을 통해 보는 공동체 회복
3. **메시아 예언**: 스가랴서에 나타난 오실 메시아의 모습
4. **언약의 갱신**: 말라기서를 통해 보는 신실하신 하나님
